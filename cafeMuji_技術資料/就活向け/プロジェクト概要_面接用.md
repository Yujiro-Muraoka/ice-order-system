## 制作物概要資料

**開発者**: 村岡 優次郎  
**開発期間**: 2025年6月〜2025年10月  

---

## 1. プロジェクト概要

**cafe&meal MUJI の店舗業務を効率化する注文管理Webアプリケーション**です。  
フード・アイスクリーム・かき氷の注文を一元管理し、レジ・キッチン・各担当者がリアルタイムに注文状況を共有できるシステムを開発しました。

## 2. 開発背景

### 解決したい課題
- 紙や口頭での注文伝達による伝達ミス
- 商品カテゴリごとに管理方法が異なる運用負担
- 繁忙時の注文進捗の共有の難しさ

### アプローチ
- 役割ごとに最適化した専用画面の設計
- クリップ色・番号による現場の運用ルールへの準拠
- Render.com へのデプロイによる複数端末からの同時利用

## 3. システム構成

- **フレームワーク**: Django 5.2 + Python 3.13
- **データベース**: SQLite / PostgreSQL
- **デプロイ**: Render.com
- **構成**: food / ice / shavedice / api

## 5. 主な機能

- フード：仮注文→本注文、キッチン進捗管理
- アイス：12フレーバー、OK/STOP/保留の状態制御
- かき氷：注文受付、待ち時間表示

## 6. 技術スタック

Python 3.13 / Django 5.2 / DRF 3.14 / Render.com
